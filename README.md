# 峰语真言

**纪念张雪峰老师**的开源公益项目。

**我为人人，人人为我** —— 用真实学生就读体验 + AI（RAG），打破教育信息差，让每一个家庭都能平等获取专业真话。

## 项目理念
每一位大学生贡献自己的专业真实感受，  
AI严格基于众包语料回答高考生和家长的疑问，  
实现教育信息平权。

不粉饰、不回避，只说真话。

## 如何参与
- 提交你的专业就读体验
- 贡献代码或改进 RAG
- 一起维护这个公益项目
